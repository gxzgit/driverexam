# driverexam
J2SE模仿驾考理论考试，布局大多数为BorderLayout和CardLayout，用来练手的

#功能大致如下
Ⅰ、一开始只有一个登录按钮，其中输入框已经设置为只能输入数字，输入别的不会有效，且最多只能输入18个数字。随意输入错误的身份证，则会有红色提示，此时也是只有一个登录按钮
输入正确，则产生开始按钮，可以进行考试

Ⅱ、错题以红色Button标记，并且将原来的1，2，…变为“X”标记，正确的题以蓝色和“√”相对应。考生在返回之前的题目时只能看，不能改（可以看到图上单选按钮变灰色），下面的四个按钮此时点击是无效果的


Ⅲ、为了美观，如果选项是四个的，则下面ABCD四个按钮全部可点击，如果只有正确和错误两个选项，则下面只有AB两个按钮可点击

Ⅳ、当答题错误时，会弹出提示框（五秒钟后自动消失），也可以手动点关闭

Ⅴ、时间到了会弹出提示，此时点击确定后，效果和交卷是一样的。

Ⅵ、交卷之后的界面比较简单，如果分数低于90，结果如上图如果高于90，分数这一栏也变为对应的分数，下面的提示改为“恭喜你，通过了科目一考试”

Ⅶ、考生参加补考时(即第二次考试)，则之前错误的题会再次出现，考生信息这一栏分数也变为之前一次的分数

Ⅷ、点击交卷，会有提示，单击否则继续考试，单击确定就交卷，直接进入界面三；如右图考生答完最后一题时会有提示考生交卷的消息弹出
